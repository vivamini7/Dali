# Dalibaba Play Store 준비 문서

이 문서는 Google Play Console 등록 화면에 입력할 내용을 미리 정리한 초안입니다.

## 기본 정보

- 앱 이름: Dalibaba
- 앱 유형: 앱
- 가격: 무료
- 카테고리 후보: 여행 및 지역정보 또는 도구
- 기본 언어: 한국어
- 웹앱 URL: https://dalibaba-app.vercel.app/
- 개인정보처리방침: https://dalibaba-app.vercel.app/privacy
- 이용약관: https://dalibaba-app.vercel.app/terms

## 짧은 설명

사진 속 메뉴판, 영수증, 문서를 인식해 번역하고 정리해주는 여행용 AI 스캔 앱입니다.

## 자세한 설명

Dalibaba는 해외 여행이나 일상에서 마주치는 메뉴판, 영수증, 안내문, 라벨 등을 사진으로 촬영해 내용을 이해하기 쉽게 정리해주는 앱입니다.

주요 기능:

- 메뉴판 사진을 인식해 메뉴명, 설명, 가격을 보기 쉽게 정리
- 영수증 사진을 인식해 항목, 금액, 합계를 빌지 형태로 정리
- 일반 문서나 라벨은 원문 텍스트를 인식하고 번역
- Google, Kakao, Naver 로그인 지원
- 모바일 앱과 웹에서 사용 가능

Dalibaba는 여행 중 언어 장벽을 줄이고, 사진 속 정보를 빠르게 확인할 수 있도록 돕습니다.

## 앱 액세스

- 로그인 필요 여부: 일부 기능은 로그인 필요
- 테스트 계정: 내부 테스트 전용 계정을 만들거나 소셜 로그인으로 테스트
- 제한된 접근: 없음

## 광고

- 현재 광고 포함 여부: 아니오

## 데이터 보안 초안

수집 또는 처리 가능성이 있는 데이터:

- 이메일 주소: 로그인 및 계정 식별
- 이름 또는 닉네임: 소셜 로그인 프로필 표시
- 사진 또는 이미지: 메뉴판, 영수증, 문서 분석
- 앱 활동: 이미지 분석 요청, 사용량 제한 처리
- 기기 또는 기타 ID: 로그인 세션, 보안 처리에 사용될 수 있음

사용 목적:

- 앱 기능 제공
- 계정 관리
- 사기 및 남용 방지
- 서비스 안정성 유지

제3자 공유 가능성:

- Supabase: 로그인 및 계정 관리
- Render: 백엔드 API 호스팅
- Vercel: 프론트엔드 호스팅
- Groq: 이미지 분석 및 텍스트 처리
- DeepL: 번역 기능
- Google/Kakao/Naver: 소셜 로그인

주의:

- Play Console 데이터 보안 답변은 실제 설정과 일치해야 합니다.
- 이미지 분석에 사용되는 사진은 개인정보가 포함될 수 있으므로 개인정보처리방침에 명시해야 합니다.

## 권한 설명

카메라:

- 사용자가 메뉴판, 영수증, 문서 사진을 촬영해 분석할 수 있도록 사용합니다.

사진/파일 접근:

- 사용자가 기존 사진을 선택해 분석할 수 있도록 사용합니다.

인터넷:

- 로그인, 이미지 분석, 번역, 서버 통신에 필요합니다.
